COVID-19 Dashboard for SI 507

The objective of this project is to aggregate data from the COVID-19 Tracking API and contextualize it with state-level statistics from the Kaiser Family Foundation. The COVID-19 API is updated every evening around 6:00PM so this dashboard should reflect updated information at that time. However, the case counts are most likely undercounts so please confer with multiple sources for accurate totals. Recent relevant Tweets and headlines are also made available through the Twitter developer's API and the NewsAPI.
